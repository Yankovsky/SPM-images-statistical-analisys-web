About

Here is the tool which you can use for statistical analysis of different images (originally for AFM topography images).

Images should be uploaded in ASCII files, which for example can be produced by Gwyddion. Top menu will guide you in uploading your matrix, choosing best filter for your image and adjusting threshold in the filtered image for finding objects which have to be analyzed. Than you can download information about lateral sizes (in pixels), height (difference between base plane and maximum of object contrast) and nearest neighbor. To know better about program please read Statistical analysis of AFM topographic images of self-assembled quantum dots by Sevriuk, V.A. and Brunkov, P.N. and Shalnev, I.V. and Gutkin, A.A. and Klimko, G.V. and Gronin, S.V. and Sorokin, S.V. and Konnikov, S.G. Semiconductors, Volume 47, 2013.

If you used this program please cite this paper and also give the acknowledgement to A. V. Yankovsky.

Topo matrices upload

                Uploaded topo matrices should be in following format:
                1	2	3
                0	0	0
                -3	2	10
                Notice, that "tab" character is used as column delimiter and '.' dot character is used as decimal mark
